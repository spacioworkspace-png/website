## Image Specifications:

- **Format**: JPG or PNG
- **Recommended Size**: 1200x800px or larger (maintain aspect ratio)
- **Optimization**: Compress images for web (use tools like TinyPNG or ImageOptim)
- **Naming**: Use lowercase with hyphens (as shown above)

## Usage:

These images are automatically referenced in:
- Homepage hero section
- Workspace cards
- Gallery page
- Location sections
- Vastu/About sections

If an image is missing, the code will gracefully fall back to placeholder icons.
